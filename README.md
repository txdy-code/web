# GraceTrail Official Website

Official website for GraceTrail - A compact spiritual companion app.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
├── index.html
├── package.json
├── vite.config.js
```

## Features

- Modern Vue 3 + Vite setup
- Responsive design
- Smooth animations
- Screenshot gallery
- Download section for App Store and Google Play

## Development

The website showcases the GraceTrail app with:
- Hero section with app introduction
- Key features overview
- Screenshot gallery
- Download call-to-action
- Footer with links

To customize, edit the Vue components in `src/components/`.
